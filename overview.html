Model Training - Overview


For developing a prediction model to predict safe duration for crossing road from a given video, we have performed many training experiments as shown below. Since road crossing for visually impaired people is an application which requires high precision in order to be deployed in real-time, we have used mean average precision (mAP) as the evaluation metric for the trained models.

Approach 1: Individual frames based approach

  • Approach 1.1

    mAP : 0.58

  • It is our simplest approach where we extracted simple per-frame features capturing number, location and size of vehicles. We used SVM to train the classification model.




  • Read more

  • Approach 1.2

    mAP : 0.64

  • It is an advancement over Approach 1.1 where we have improved our feature extraction logic by ignoring the vehicles traveling on the opposite half of the road (using vehicle tracking).



  • Read more

  • Approach 1.3

    mAP : 0.74

  • It is an advancement over Approach 1.2 where we have improved our feature extraction logic by considereing relative speed of the vehicles, and we improved the labels by annotating the videos frame-wise (instead of second-wise).

Approach 2: Multiple frames based approach

  • Approach 2.1

    mAP : 0.80

  • As it is obvious that even we as humans do not decide whether it is safe to cross a road by just having one glance at the road, we have started using multi-frame features instead of individual-frame features in this approach.

    mAP : 0.80

  • Approach 2.2 is similar to Approach 2.1, in which we have used multi-frame features in a sliding window based manner. Its feature extraction logic is a bit different as compared to Approach 2.1.
